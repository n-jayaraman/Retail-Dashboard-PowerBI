This is my first official project to be pushed in Git.This project shows the retail shops details of Malls in Istanbul,Turkey.
DATAS:
It includes the following columns:
-Invoice-ID
-Customer-ID
- Gender
- Age
- Category
- Quantity
- Price
- Payment Method
- Invoice Date
- Shopping Mall
 
Tool used:
PowerBI(online)

Visuals Used:
- Donut chart (Payment Method share)
- Bar chart (Sales by category)
- KPI cards (Total Sales, Total Quantity)
- Pie chart (Customer Gender)
- Tabular data view

Key Insights:
- Most customers pay using UPI & Cash
- Category A/B contribute maximum sales
- Women customers buy slightly more than men

Future Improvement:
- Add slicers and drill-through using Power BI Desktop
